Overview

This is a supporting and healing build for Chronomancer that uses wells to heal allies while also maintaining 100% uptime on Alacrity and Quickness. This build also can give Aegis and Stability for certain mechanics that require them by delaying . Note that this build does NOT provide Might, so you will need for example a Scepter/Warhorn Tempest for that.
