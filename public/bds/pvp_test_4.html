Overview

A high damage and high survivability Power Scourge. Using carapace armour, Wells and a few spectral skills, this build can achieve high damage and tankiness. Stuns and CC are your worst enemy though.

Specialization Changes

If you want to live dangerously, you can run Spite and Soul Reaping for more damage and combos like Dread + 50% more fear duration. Spite can work with any of the above in combination but is weaker in survivability. Spite works great for maintaining Carapace stacks because of all the vulnerability you will be applying.

Tips

  • + before the start of match to build Shroud before the match begins and even during down time in the match.
  • + is pretty much a second heal skill without much investment in healing power. It's a pretty combo they made.
  • + is like a stun because the target cannot move or activate skills for 2s. So try to use them in combination to shut targets down for a second
  • Torch #5 only knocks down 1 target. Don't be deceived like I was.
  • You can activate all shroud skills whilst activating other skills so make sure you really learn how to do two things at once with this build because that's the difference between a good or bad Scourge.
